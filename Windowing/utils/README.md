# Windowing/utils

Utilidades de soporte para lectura/procesado de imagen y operaciones auxiliares usadas por el modulo de windowing.

## Contenido

- `dicom_utils.py`: helpers DICOM de bajo nivel (LUT y transformaciones relacionadas).
- `readers.py`: lectura/resampleo de imagenes (incluye utilidades con SimpleITK).
- `image.py`: funciones auxiliares de clipping/escalado.

## Alcance

Estas utilidades estan pensadas como helpers internos del paquete `Windowing/`.
